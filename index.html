My Work

Book Recommender System

The Book Recommender System provides personalized book suggestions based on user preferences and reading history. Using collaborative and content-based filtering techniques, it analyzes user ratings, genres, and authors to recommend books that match individual tastes.

RISV Pipelined CPU Design

The RISC pipelined CPU is designed to efficiently execute a sequence of instructions through a multi-stage pipeline architecture. This design enhances instruction throughput by overlapping the execution of multiple instructions, allowing for higher performance in processing tasks.

Filters and Audio Mixer

The Filters and Audio Mixer project focused on designing audio filters to meet specific performance criteria. I analyzed their characteristics, created a functional audio mixer using these filters, and assisted in circuit design by calculating expected performance data and measuring actual results.
